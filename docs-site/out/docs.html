lux-crypto

lux-crypto is a high-performance cryptographic library providing GPU-accelerated implementations of essential blockchain cryptographic primitives via Metal compute shaders for Apple Silicon.

Key Features

  • BLS12-381 Curve Operations - Pairing-friendly curve for aggregate signatures
  • BN254 Curve - Efficient curve for zk-SNARKs
  • KZG Polynomial Commitments - Polynomial commitments for data availability
  • BLAKE3 Hashing - Modern, parallelizable cryptographic hash
  • ML-DSA (Dilithium) - Post-quantum digital signatures (FIPS 204)
  • CGO Bindings - Native Go integration
