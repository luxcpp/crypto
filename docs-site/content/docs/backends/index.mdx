# Backends

lux-crypto supports multiple compute backends to leverage available hardware acceleration:

| Backend | Platform | Use Case |
|---------|----------|----------|
| **Metal** | Apple Silicon | GPU acceleration on macOS/iOS |
| **CUDA** | NVIDIA GPUs | GPU acceleration on Linux/Windows |
| **CPU** | All platforms | Fallback and compatibility |

## Backend Selection

The library automatically selects the optimal backend based on available hardware:

```cpp
#include "crypto.hpp"

// Check available backends
auto backends = lux::crypto::available_backends();
// Returns: {"metal", "cpu"} on Apple Silicon

// Force specific backend
lux::crypto::set_backend("metal");  // Use GPU
lux::crypto::set_backend("cpu");    // Use CPU only
```
